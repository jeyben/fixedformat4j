/**

21 
 *
22 
 * <p>The first call to {@link #get} for a class scans its annotations and builds one
23 
 * {@link FieldDescriptor} per effective {@code @Field}. Subsequent calls return the same
24 
 * immutable list without re-scanning.
25 
 *
26

27

29

30

31

34
